# Диаграммы для ВКР IndiGuide

Каждый файл содержит исходный код одной диаграммы и инструкцию, в какой
онлайн-редактор его вставить. Полученные изображения вставляются в текст ВКР
на места соответствующих рисунков (нумерация рисунков сквозная).

| Файл | Рисунок в ВКР | Редактор |
|------|---------------|----------|
| 01_c4_context.md       | Рисунок 1 — Контекстная диаграмма C4 (C1) | PlantUML |
| 02_c4_container.md      | Рисунок 2 — Диаграмма контейнеров C4 (C2) | PlantUML |
| 03_deployment.md        | Рисунок 3 — Диаграмма развёртывания | PlantUML |
| 04_activity_process.md  | Рисунок 4 — Диаграмма деятельности процесса | PlantUML |
| 09_er_dbdiagram.md      | Рисунок 5 — ER-диаграмма предметной области | dbdiagram.io |
| 05_functions_tree.md    | Рисунок 6 — Дерево функций пользователя | PlantUML |
| 06_sequence_generate.md | Рисунок 7 — Диаграмма последовательности «Генерация маршрута» | PlantUML |
| 07_structure.md         | Рисунок 8 — Структурная схема ИС | PlantUML |
| 08_algorithm_corridor.md| Рисунок 9 — Блок-схема алгоритма «эндпоинт-коридор» | PlantUML |

Рисунки 10–16 — это скриншоты работающего приложения; их нужно сделать
самостоятельно (см. подписи-заглушки в тексте ВКР).
